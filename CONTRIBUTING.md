# Contributing to grug-cli

grug appreciate help. grug keep simple.

## 🪨 The Grug Way

1. **No complexity** - Simple solutions only
2. **No dependencies** - stdlib only
3. **Ship fast** - Working code > perfect code

## Pull Requests

1. Fork repo
2. Make changes (keep simple!)
3. Test with `bun run index.ts`
4. Submit PR

grug say: "complexity bad, contributor good" 🪨
